# find-your-duo 🎮

App em React.js, React Native, Node.js para achar seu duo de games

## Rodando localmente 👇

Clone o projeto

```bash
  git clone https://github.com/renyzeraa/find-your-duo.git

  ou

  git@github.com:renyzeraa/find-your-duo.git
```

Entre no diretório do BackEnd

```bash
  cd server
```

Instale as dependências

```bash
  npm install
```
